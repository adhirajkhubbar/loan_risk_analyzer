# 💰 Loan Risk Analyzer

🔗 Live Demo: https://adhirajkhubbar.github.io/loan_risk_analyzer/

A smart fintech-style web app that analyzes loan approval risk based on financial inputs.

## 🚀 Features
- Loan EMI calculation
- Risk scoring system
- Approval probability
- Interactive UI with live updates
- Doughnut analytics chart

## 🧠 Tech Used
- HTML, CSS, JavaScript
- Chart.js

## 📊 How it Works
- Calculates EMI using standard formula
- Evaluates risk using:
  - EMI-to-income ratio
  - Savings
  - Credit score
  - Job stability
- Generates approval decision + probability
