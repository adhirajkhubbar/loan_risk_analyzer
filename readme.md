# 💰 Loan Risk Analyzer

[🚀 Live Demo](https://adhirajkhubbar.github.io/loan_risk_analyzer/)

---

## 📌 Overview
A fintech-style web application that analyzes loan approval risk using financial inputs like income, expenses, EMI, credit profile, and job stability.

It provides:
- Risk score
- Approval decision
- Approval probability
- EMI and loan breakdown
- Visual analytics

---

## ✨ Features

- 📊 Real-time loan risk analysis  
- ⚡ Instant UI updates (no button clicks)  
- 💰 EMI + Total repayment calculation  
- 🎯 Approval probability prediction  
- 📉 Interactive doughnut chart  
- 🎨 Modern fintech-style UI  

---

## 🧠 How It Works

The system uses a rule-based scoring model:

- **EMI-to-Income Ratio**
- **Savings after expenses**
- **Credit profile**
- **Job stability**

Each factor contributes to a **risk score**, which determines:

| Risk Score | Decision |
|-----------|----------|
| < 30      | Approved |
| 30–70     | Review |
| > 70      | Rejected |
